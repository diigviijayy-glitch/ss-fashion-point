# SS Fashion Point

Premium men's wear & shoes store in Udaipur, Rajasthan.

**Live site:** _Deploy to Vercel — see below_

## Stack

Static HTML/CSS/JS — no build step required.

## Structure

```
index.html          # Main page
css/styles.css      # All styles
js/main.js          # App logic (cart, wishlist, account, products)
js/tweaks.js        # Tweaks panel (mood/energy/edge)
assets/photos/      # Store & product photos
```

## Features

- Street-marquee hero with Ken Burns animation
- Product grid with filtering by category
- Product detail panel (colours, sizes, fits)
- Cart drawer + checkout → WhatsApp order
- Wishlist (heart-save, drawer, move-to-bag)
- Account (Google / Apple / mobile OTP sign-in)
- "Added to bag" toast notification
- Dark / light mode (persists, no flash)
- Tweaks panel: Mood (brown palettes), Energy (animation speed), Edge (corners)
- Fully responsive — mobile-first

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Framework: **Other** (static site)
4. Root directory: `/` (default)
5. Deploy — done. No build command needed.
